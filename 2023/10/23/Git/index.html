Git

2023/10/23

Git基本命令
git init 开始Git旅程
git add filename 添加文件到缓存区,filename填写文件名，要加后缀哦
git add . add所有文件
git rm –cached filename 去除缓存区文件
git status 查看缓存区内容
git commit -m “anything” 提交本地仓库，anything填写备注
git push -u origin master 本地库推送到远程库,-u在第一次推送时使用,origin作别名使用可自定义
git push -u之后可如此简化使用，-u作用是把本地和远程仓库关联
git log 查看当前文件版本
git reset –hard ID 恢复版本，ID可自定义也可在log中查看

Git与远程仓库关联
git remote add origin server 关联远程仓库 server填github的SSH
git remote remove origin 解除关联

Git分支

这非常有意义！
在多人协作时，可以隔离工程，各自工作负责的内容，最后再进行合并

Git分支命令
git branch 查看分支
git branch xxx 创建分支，xxx分支名
git branch -d xxx 删除分支
git checkout xxx 切换分支
git checkout -b xxx 快速创建和切换
git merge xxx 合并分支x，注意：是将x合并至当前分支
git remote show xxx 查看远程分支
git push origin xxx 推送至远程
git push origin –delete xxx 删除远程分支
git pull 更新当前分支

实例演示

使用git branch查看当前所有的分支，图示当前为test
使用git checkout master切换到master主分支

再次查看，成功切换到主分支
执行git branch -d test删除test，并使用git branch test新建test

最后将新建的test推送到远程库中，命名远程分支test

可以见到GitHub中更新了test
