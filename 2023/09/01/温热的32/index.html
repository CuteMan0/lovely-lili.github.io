误差

adc读出来温度居然有120°C？？，怀疑是取错地址了，结果是没有自校准

/********************ADC*********************/
int ADCtemp()
{
    float temp = 0.0;
    u16 TS_DATA,TS_CAL1,TS_CAL2;
    
    TS_CAL1 = *(u16*)(0x1FF1E820);
    TS_CAL2 = *(u16*)(0x1FF1E840);
    
    HAL_ADC_Start (&hadc3);                             //开采
    if(HAL_ADC_PollForConversion(&hadc3,10)==HAL_OK)    //等货
    {
        TS_DATA =HAL_ADC_GetValue(&hadc3);              //取货
    }
    HAL_ADC_Stop(&hadc3);                               //完毕
    
    temp = (80 * (TS_DATA - TS_CAL1)) / (TS_CAL2 - TS_CAL1) + 30;

    LCD_DisplayDecimals(80,180,temp,8,2);               //在LCD上显示

    return temp ;
}

ADC自校准

在HAL_ADC_Init后，调用此函数启动ADC的自校准功能，支持偏移校准和线性度校准

HAL_ADCEx_Calibration_Start(&hadc3,ADC_CALIB_OFFSET,ADC_SINGLE_ENDED);
参数、返回值
ADC_CALIB_OFFSET 只运行偏移校准而不运行线性度校准
ADC_CALIB_OFFSET_LINEARITY 同时运行偏移校准和线性度校准
ADC_SINGLE_ENDED 单端模式
ADC_DIFFERENTIAL_ENDED 差分模式
HAL_ERROR 配置失败
HAL_OK 配置成功
HAL_BUSY 操作中
HAL_TIMEOUT 时间溢出

说明：
偏移校准 ： 测量 ADC 参考电压的实际值，将该值与 预设的参考电压 比较，得到参考电压的偏差，校准偏移量，为了准确 测量 ADC 输入信号。
线性度校准 ： 测量 内部基准电压 和 ADC 输入信号的幅值之间的比例关系，校准 ADC 增益的方式，确保 ADC 输出的数值与输入信号的幅值之间具有良好的线性关系，为了准确 转换 ADC 输入信号
注：
线性度校准可能会出现 timeout ，修改 ADC_CALIBRATION_TIMEOUT 的值
