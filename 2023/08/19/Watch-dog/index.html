зңӢй—ЁзӢ—

е…¶е®һе°ұжҳҜдёҖдёӘе®ҡж—¶еҷЁпјҢеңЁзЁӢеәҸ и·‘йЈһ/жӯ»еҫӘзҺҜ ж—¶ еӨҚдҪҚ

IWDG е® зү©зӢ—


дёҖдёӘ 12bit зҡ„йҖ’еҮҸи®Ўж•°еҷЁпјҢи®Ўж•°еҷЁеҖјдёә 0 ж—¶пјҢдә§з”ҹдёҖдёӘ IWDG_RESET еӨҚдҪҚдҝЎеҸ·
е–ӮзӢ—пјҡеңЁзӢ—еҸ«д№ӢеүҚеҲ·ж–°и®Ўж•°еҖј

зӣёе…іж—¶й—ҙи®Ўз®—
Tiwdg = (4*2^prv) / LSI * rlv (s)
жіЁпјҡprv ---- йў„еҲҶйў‘еҷЁеҖјпјҲ8bitпјү     rlv ---- йҮҚиҪҪеҜ„еӯҳеҷЁеҖј
E.gпјҡLSI = 40KпјҢ prv = 256пјҢrlv = 625пјҢеҲҷTiwdg = 4s

WWDG иӯҰзҠ¬

дёҖдёӘ 7bit зҡ„йҖ’еҮҸи®Ўж•°еҷЁ еҲ° еӣәе®ҡеҖј 0X40 ж—¶дёҚе–ӮзӢ—пјҢдә§з”ҹеӨҚдҪҚпјҢиҝҷдёӘеҖјеҸ« зӘ—еҸЈзҡ„дёӢйҷҗ пјҢеӣәе®ҡдёҚеҸҜеҸҳ
и®Ўж•°еҷЁеҖјеңЁеҮҸеҲ°жҹҗдёҖдёӘж•°д№ӢеүҚе–ӮзӢ—зҡ„иҜқд№ҹдјҡдә§з”ҹеӨҚдҪҚпјҢиҝҷдёӘеҖјеҸ« зӘ—еҸЈзҡ„дёҠйҷҗ пјҢдёҠйҷҗеҖјз”ұз”ЁжҲ·зӢ¬з«Ӣи®ҫзҪ®

зӣёе…іж—¶й—ҙи®Ўз®—
Twwdg = Tpclk1 x 4096 x 2^wdgtb x (T[5:0] + 1) (ms)
жіЁпјҡTpclk1 ---- APB1ж—¶й’ҹе‘ЁжңҹпјҢ 2^wdgtb ---- йў„еҲҶйў‘еҷЁпјҲ4bitпјү 
    T[5:0] ---- жҲ‘жүҖзҗҶи§ЈпјҡT6дҪҚзҪ®0ж—¶пјҢWWDGеӨҚдҪҚпјҢжүҖд»ҘдёҠйҷҗеҖјзҡ„T6дҪҚйғҪжҳҜзҪ®1зҡ„пјҢдёҠдёӢйҷҗе·®еҖјдёә0x00-0x3F
E.gпјҡTpclk1 = 1/36MпјҢ wdgtb = 0, T[5:0] = 0x00, еҲҷTwwdg = 113Ојs
