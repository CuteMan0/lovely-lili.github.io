SG90小记

2023/8/17 MCU

为什么写SG90

一些机缘巧合居然又用到它啦，SG90是很早就使用过的一款舵机，发现有些遗忘，特此做个备份

接线方式

红色 ---------------------- VCC
棕色 ---------------------- GND
橙色 ---------------------- PWM

工作原理

SG90的控制信号是周期 20 ms （即频率50Hz）的脉宽调制（PWM）信号，通过调节脉冲宽度驱动舵机
以下是 180°版本的脉宽与角度关系：

脉宽                舵机转角度              占空比
0.5 ms --------------   0°  --------------  2.5%
1.0 ms --------------  45°  --------------   5%
1.5 ms --------------  90°  --------------  7.5%
2.0 ms --------------  135° --------------  10%
2.5 ms --------------  180° --------------  12.5%

而360°版本因为无角度定位，上电并不会复位到0°位置，此时调节脉宽控制它的旋转

    脉宽                舵机状态                占空比
0.5-1.5 ms -------------- 正转 --------------  2.5-7.5%
1.5-2.5 ms -------------- 反转 --------------  7.5-12.5%
  约1.5 ms -------------- 暂停 --------------    7.5%
  约2.5 ms -------------- 转速 --------------   12.5%

不同的舵机可能有不同的中位，所以并不一定是在1.5/2.5 ms暂停/转速调节！会有一定的偏差，需要实际测试捏..

代码

void TIM5_PWM_Init(u16 arr,u16 psc)
{
    //定义初始化结构体
    GPIO_InitTypeDef GPIO_InitStructure;
    TIM_TimeBaseInitTypeDef TIM_TimeBaseStructure;
    TIM_OCInitTypeDef TIM_OCInitStructure;
    
    //使能定时器时钟 TIM5
    RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM5, ENABLE);
    //使能GPIOA外设时钟使能
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA, ENABLE);
    //初始化GPIO
    //设置复用输出功能TIM5
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_0; //TIM5 PA0
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AF_PP; //复用推挽输出
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(GPIOA, &GPIO_InitStructure);
    
    //初始化时基
    //arr = 9999
    TIM_TimeBaseStructure.TIM_Period = arr;
    //psc = 143
    TIM_TimeBaseStructure.TIM_Prescaler =psc;
    TIM_TimeBaseStructure.TIM_ClockDivision = 0;
    TIM_TimeBaseStructure.TIM_CounterMode = TIM_CounterMode_Up;
    TIM_TimeBaseInit(TIM5, &TIM_TimeBaseStructure); 
    
    //输出模式配置
    TIM_OCInitStructure.TIM_OCMode = TIM_OCMode_PWM1;
    TIM_OCInitStructure.TIM_OutputState = TIM_OutputState_Enable;
    TIM_OCInitStructure.TIM_Pulse = 0;
    TIM_OCInitStructure.TIM_OCPolarity = TIM_OCPolarity_High;
    TIM_OC1Init(TIM5, &TIM_OCInitStructure);
    
    TIM_CtrlPWMOutputs(TIM5,ENABLE);
    
    TIM_OC1PreloadConfig(TIM5, TIM_OCPreload_Enable);
    TIM_OC2Init(TIM5, &TIM_OCInitStructure);
    
    //使能ARR、MOE及定时器
    TIM_ARRPreloadConfig(TIM5, ENABLE); //使能TIMx在ARR上的预装载寄存器
    TIM_Cmd(TIM5, ENABLE); //使能TIM5
}

void SetAngle(float angle)
{
    angle=(u16)(50.0*angle/9.0+249.0);
    TIM_SetCompare1(TIM5,angle);
}
